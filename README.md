# Temp

이 레파지토리는 "클론용"이예요.
새로운 프로젝트 빌드를 쉽게 만들어줘요.

이 프로젝트는 기본적으로 다음 사항을 가져요.

1. 타입스크립트
2. 리액트
3. vite
4. 라우터
5. 테일윈드
6. yarn
7. MIT 라이센스

## 빠른 시작

### 1. 클론 및 폴더명 변경

.. temp.git 뒤에 한 칸 띄우고 원하는 프로젝트 이름을 쓰세요.

```bash
git clone https://github.com/kss2002/Tailwind_temp.git [원하는_프로젝트_이름]
cd [원하는_프로젝트명_이름]
```

### 2. 설치

```bash
yarn install
```

### 3. 실행

```bash
yarn dev
```

개발 서버는 http://localhost:5173 이예요.

### 요구사항

- Node.js 14+
- yarn

**클론 이후 리드미는 변경해주세요!**
